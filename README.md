## 🛠 기술 스택
- **Framework:** Vue 3
- **Build Tool:** Vite
- **Routing:** Vue Router
- **API:** TMDB API (Axios)
- **Deployment:** Netlify

## ✨ 주요 기능
1. **홈 화면**: 인기, 최신, 평점순 영화 리스트 제공 (Infinite Carousel)
2. **상세 페이지**: 영화 상세 정보 및 공식 예고편(YouTube) 재생
3. **찜하기(Wishlist)**: 하트 버튼을 통한 관심 영화 저장/삭제 (LocalStorage 연동)
4. **검색 기능**: 키워드 기반 영화 검색
5. **UI 디자인**: Glassmorphism 및 네온 효과를 적용한 사이버펑크 테마
